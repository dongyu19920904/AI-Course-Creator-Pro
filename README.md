## 🖱️ 新手操作指南
### 如何查看课程？
1. 直接点击上方蓝色文字链接（比如 [Day1 认知重启](Day1-认知重启.md)）
2. 页面打开后点击右上角的「Download」按钮即可保存到手机/电脑
3. 推荐用「Typora」或「有道云笔记」打开.md文件，阅读体验更佳

### 电脑端用户
1. 点击页面绿色「Code」按钮
2. 选择「Download ZIP」下载全套课程
3. 解压后双击打开想学的课程文件

### 手机端用户
1. 直接点击课程目录中的链接
2. 进入页面后点击右上角「···」选择「在浏览器打开」
3. 滑动屏幕即可流畅阅读
